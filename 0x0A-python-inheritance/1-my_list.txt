=================
Print list sorted
=================

Prints a list in sorted manner

---------------
Import function
---------------
    >>> MyList = __import__('1-my_list').MyList

----------------
normal operation
----------------
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]

-------------------
Print sorted normal
-------------------
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

---------------
only one number
---------------
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1]

-----------
no argument
-----------
    >>> my_list = MyList()
    >>> my_list.print_sorted()
    []

----------------
Repeated Numbers
----------------
    >>> my_list += [1, 2, 5, 7, 9, 5, 2]
    >>> my_list.print_sorted()
    [1, 2, 2, 5, 5, 7, 9]

----------------
Negative Numbers
----------------
    >>> my_list = MyList()
    >>> my_list += [1, 2, -5, -7, 9, 5, -2]
    >>> my_list.print_sorted()
    [-7, -5, -2, 1, 2, 5, 9]

-------------------------
is instance of class list
-------------------------
    >>> isinstance(my_list, list)
    True

-------------------------------
is a subclass of the class list
-------------------------------
    >>> issubclass(MyList, list)
    True

---------------------------
initialize with an argument
---------------------------
    >>> my_list = MyList([1, 2])
    >>> my_list.print_sorted()
    [1, 2]

    ---
    two
    ---
    >>> my_list = MyList([1, 2], [5, 6])

    ...
    TypeError: list expected at most 1 argument, got 2

------------------------------
    >>> my_list = MyList(5)

-----------
pass string
-----------
    >>> my_list = MyList("string")

    ['g', 'i', 'n', 'r', 's', 't']

---------------------------
